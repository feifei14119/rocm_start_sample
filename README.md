Rocm Start Sample
====
rocm-hip on amdgpu入门示例

---
## 1. 使用：
run  
**"python build.py [*hip/asm*] [*v2/v3*] [*llvm/hcc*]"**  
under each project folder, the exacutable will generated under ./out path.
